while( tempQ != "000...")
{
	for(int i=0;i<tempQ.length;i++) // 从左到右遍历tempQ,一个零寄存器作为余数
	{
        将余数乘以10
        将当前字符转换为对应数字，加到余数上
        计算余数/2的商，转换为字符，添加到temp字符串，该字符串存本次循环遍历的商

                如果商为0,将temp填满0
                不为0则与正常情况一同继续处理
            如果商为0，则直接跳过，不添加到temp字符串 
        更新余数为余数%2
	}
    将余数转换为字符，添加到结果字符串(insert到最前面，或者最后逆序)
    将temp放进tempQ
    若tempQ字符串不为0，将tempQ字符串作为被除数继续计算，直到tempQ字符全为0
